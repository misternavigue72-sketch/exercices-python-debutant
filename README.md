# 🐍 Mes Projets Python Personnels

Ce dépôt regroupe une série de défis de programmation que j'ai réalisé en autonomie. L'objectif est de mettre en pratique les concepts fondamentaux du langage **Python** tout en explorant des projets concrets.

## 📁 Projets réalisés (Niveau Débutant)

Voici la liste des outils et jeux développés dans ce répertoire :

* **Mad Libs Generator** : Un jeu de création d'histoires aléatoires avec gestion des entrées utilisateur (en cours de finalisation).
* **Quiz Game** : Un jeu de questions-réponses avec calcul du score final.
* **Bill Splitter** : Un utilitaire pour diviser équitablement des factures entre plusieurs personnes.
* **Calculator** : Une calculatrice gérant les opérations arithmétiques de base.
* **Countdown Timer** : Un minuteur utilisant le module `time`.
* **Dice Roller** : Un simulateur de lancer de dés basé sur le module `random`.
* **Number Guessing Games** : Un jeu où l'utilisateur doit deviner un nombre secret choisi par l'ordinateur.
* **To Do List** : Un gestionnaire de tâches simple pour s'exercer à la manipulation des listes.

## 🛠️ Environnement Technique

* **Système** : Linux Ubuntu (Dual Boot Win/Ubuntu)
* **Matériel** : Dell Latitude 5400
* **Éditeur** : VS Code
* **Langages** : Python 3.12
